# Cinematic Birthday Surprise 🎉

A beautiful, cinematic, and interactive web page created as a special birthday surprise. 

## Features
- **Neon-themed Hero Section**: A stylish entrance with glowing buttons and typography.
- **Interactive Message Slider**: A step-by-step unboxing of the surprise.
- **Dynamic Fairy Lights & Balloons**: Magical animations that bring the celebration to life.
- **Memory Gallery**: A beautiful grid showcasing favorite memories and photos.

## Tech Stack
- **HTML5**: For semantic structure.
- **Vanilla CSS3**: Covering custom animations, keyframes, styling, variables, and responsive design. No external frameworks used.
- **Vanilla JavaScript**: Handling DOM manipulation, interactive sequences, dynamic particles, and state transitions.
